# Employee Management System (SQL)

## Project Overview
This project demonstrates basic SQL operations using an Employee Management System.

## Features
- Create Employee Table
- Insert Employee Records
- Retrieve Employee Data
- Filter Data using WHERE
- Sort Data using ORDER BY
- Calculate Highest and Average Salary
- Count Employees
- Group Employees by Department

## Technologies Used
- SQL
- MySQL
- DB Fiddle

## SQL Concepts Covered
- CREATE TABLE
- INSERT
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- COUNT
- AVG
- MAX
